# spectra-vm
* Spectra deployment, testing, and installers.
* minimal requirement deployment for commodity hardware

# Requirements
* QEMU, amd64, i686, most supported archs
* qemu imgs can also be remotely built

# VM Deploys

OS | x86 (32) | AMD64
------------ | ------------- | ----
Linux | Y | Y
Win | N | N
OSX | N | N

# Install
* linux x86 - installi32.sh
